## 7. Kesimpulan Interaksi

### Flow Utama Aplikasi:
1. **MainActivity** â†’ Entry point, memanggil `MainAppContent()`
2. **App.kt** â†’ Central hub untuk navigasi dan state
3. **MainViewModel** â†’ Data persistence dan business logic
4. **Navigation.kt** â†’ Route definitions
5. **Screen files** â†’ UI untuk setiap halaman
6. **Components** â†’ Reusable UI pieces
7. **Theme** â†’ Consistent styling

### Prinsip Desain:
- **Unidirectional Data Flow**: State turun, events naik
- **Single Source of Truth**: ViewModel holds data
- **Composition over Inheritance**: Small composables
- **Separation of Concerns**: UI, logic, navigation terpisah
